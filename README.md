# Stock Weekly K-Line Dashboard

A lightweight, chart-focused dashboard to monitor weekly K-line charts for Chinese stocks (A-share & Beijing Stock Exchange).

## Features
- **Multi-Chart Grid**: View 20+ charts simultaneously.
- **Support for All Markets**: Shanghai (`6xxxx`), Shenzhen (`0xxxx`/`3xxxx`), and Beijing (`8xxxx`/`4xxxx`/`920xxx`).
- **Excel Import**: Drag & Drop your `.xlsx` file (column `股票代码`) to load up to 100 stocks instantly.
- **Favorites**: Star your best stocks. They are saved in your browser and auto-load next time.
- **Export**: Backup your favorites to an Excel file.
- **Responsive**: Auto-adjusts columns, or set a fixed number of columns for your big screen.

## How to Use
1. Open the [Live Dashboard](https://<YOUR-USERNAME>.github.io/<YOUR-REPO-NAME>/).
2. The default list loads automatically.
3. Drag your `stocks-example.xlsx` onto the page to import your watch list.
4. Click the ★ to save a stock to your favorites.

## Deployment
Just push this directory to a GitHub repository and enable **GitHub Pages** (Source: `main` branch).
